내보내기
스케치에서 아이콘, 모바일 앱, 웹 사이트 또는 다른 어떤 것을 디자인 할 때, 작업물의 일부 또는 전부를 내보낼 수 있습니다! 스케치를 사용하면 내보내려는 항목과 내보내는 방법을 간단히 수행 할 수 있습니다.
다른 팀원들에게 작업물을 공유하려면 메뉴에서 Share > Export...를 선택하거나(Shift-Command-E) 툴바에서 내보내기 항목을 클릭하십시오. 스케치의 캔버스는 영역이 무한하기 때문에 내보낼 부분을 따로 정해줘야합니다.

내보내기를 클릭하면 스케치가 아트 보드, 슬라이스 또는 일반 레이어 등 캔버스의 내보낼 수있는 모든 레이어를 나열하는 대화 상자를 표시합니다. 목록에서 선택하거나 선택 취소하여 내보낼 항목을 선택할 수 있습니다.
아래 비디오에서 Slice와 Exportable Layers의 차이점과 스케치에서 내보내기에 대한 일반적인 개요를 볼 수 있습니다.
Exportable하게 만들기
스케치에서 레이어, 그룹 또는 아트보드를 Exportable Layers로 표시하려면, 해당 항목이 선택되어 있는지 확인하고 Inspector 아래 쪽에 있는 "Exportable" 버튼을 클릭하십시오.
레이어가 'Exportable'으로 설정되면 버튼 부분에 새로운 뷰가 나타납니다. 내보내기 설정을 조정 할 수 있는 뷰로, 여기서 설정한 항목은 레이어를 내보낼 때 적용됩니다.

"Export" 제목 오른쪽의 + 버튼을 클릭하면 다른 크기로 내보내는 설정이 추가됩니다. 기본값으로 이전에 했던 설정대로 내보내며, 이 영역에서 따로 설정할 경우 설정한대로 이미지를 내보냅니다.

Preference 아이콘을 사용하면 미리 정해놓은 여러 사전설정을 쉽게 불러올 수 있습니다. 팝업 메뉴에서 기존의 사전 설정을 만들고 편집할 수 있는 옵션도 여기에서 찾을 수 있습니다. 내보내기 사전 설정에 대한 자세한 내용은 사전 설정 내보내기을 참조하십시오.
내보내기 설정
내보내기 설정은 모든 레이어 또는 슬라이스에 적용되며 캔버스에서 따로 지정하지 않아도 여러 크기 또는 파일 유형으로 내보낼 수 있습니다. 여기서 세 가지 주요 컨트롤이 표시됩니다. 크기, 접두사/접미사, 포맷
크기
값을 입력하거나 팝업 메뉴에서 값을 선택하여 요소가 내보내는 크기 또는 배율을 정의 할 수 있습니다.

기본 크기는 '1x'입니다. 이 설정으로는 만들어진 레이어의 실제 크기로 레이어를 내보냅니다. 모바일 용으로 디자인하는 경우 원본의 2배 크기가 필요할 때 '2x'로 내보낼 수 있습니다. 얼마나 큰 비율로 할 수 있는지에 대한 제한이 없으며 '0.5x'와 같이 축소 할 수도 있습니다. Android 기기를 위해 설계한 경우 맞춤 크기로 내보낼 수 있습니다.

배율을 곱하는 것 외에도 특정 너비 또는 높이로 내보낼 수 있습니다. 예를 들어 원래 크기에서 128 픽셀 너비로 레이어를 내보내려면 '128w'를 입력하면됩니다. '128h'로 작성할 경우 128px 높이로 내보냅니다.

픽셀 밀도에 대해 알아야 할 모든 것을 배우려면 Peter Nowell의 Pixel Density, Demystified를 확인하십시오.
접두어/접미사
이 부분은 내보낼 때의 크기 조절과 함께 작동합니다. 내보내기 배율이 두 개 이상일 경우 파일의 이름 앞이나 뒷부분에 하나 이상의 접두사 또는 접미사를 추가해야 합니다.

텍스트 필드의 인디케이터를 클릭하고 팝업 메뉴를 통해 옵션을 선택하여 내보내기 축척에 접두사/접미사를 적용할지 여부를 선택할 수 있습니다. 필드의 줄임표 (...)는 레이어 이름을 나타냅니다. 그 앞에 나타나는 것은 접두어이며 끝에는 파일 포맷 앞에 접미어가 나타납니다.

일반적으로 '2x'스케일로 내 보낸 객체는 "@ 2x"의 접미사를 갖습니다. 이는 OS를 설계 할 때 Apple이 요구하는 이름 지정 규칙이기 때문입니다. 그러나 Android 용으로 디자인하는 경우 '2x'는 'xhdpi /'라는 접두어로 표시됩니다.
포맷
format 버튼을 사용하여 팝업 메뉴를 통해 내보낼 파일 포맷을 선택할 수 있습니다. 파일 포맷은 아래에 있는 섹션에서 더 자세히 알아볼 수 있습니다.
내보낼 수 있는 레이어나 슬라이스
레이어 내보내기는 Canvas의 다른 요소에 상관없이 해당 레이어만 내보내는 것입니다. 해당 레이어의 위/아래에 다른 레이어가 있어도 내보내기에 포함되지 않습니다.

레이어를 내보낼 수 있는 것으로 표시하면 대부분의 상황에서 사용할 수 있지만 모든 상황에 대응할 수 있지는 않습니다. 이 때 슬라이스가 필요합니다.

내보낼 수 있는 레이어에서 바로 슬라이스를 만들 수 있습니다. "Export" 타이틀의 슬라이스 버튼을 클릭하기만하면 설정이 포함된 오브젝트 주위에 새로운 슬라이스 레이어가 만들어집니다.

레이어가 슬라이스되었거나 내보내기가 가능한지 여부에 관계없이 Command-E 바로 가기를 통해 빠르게 내보낼 수 있습니다.
레이어 목록
내보낼 수 있게 만들어진 레이어는 레이어 목록에서 작은 슬라이스 아이콘이 표시됩니다.
따로 작업하지 않아도 레이어 목록에서 바로 레이어를 직접 내보낼 수 있습니다. 애플리케이션의 레이어 목록에서 레이어를 클릭하고 드래그하면 PNG로 신속하게 내 보냅니다. Option 키를 누르고 있으면 PDF로 내보내집니다.
공유하기
개체를 내보내려면 "Export ..."버튼를 클릭하여 컴퓨터의 위치에 빠르게 저장하거나 공유 버튼를 클릭하여 공유할 수 있습니다. 메일 첨부, 트윗, Airdrop 등
슬라이스
캔버스에서 이미지를 내보내는 방법 중 하나입니다.
슬라이스는 일반 레이어와는 다른 개념으로 다른 레이어와 함께 선택, 크기 조정, 이동 및 그룹화 할 수 있습니다. 슬라이스를 따로 숨길 수 있으며 레이어 목록의 맨 아래에 있는 슬라이스 버튼를 토글하여 문서 내에서 모든 슬라이스를 숨기거나 표시 할 수 있습니다.
슬라이스 추가하기
슬라이스를 만들려면, 팝업 메뉴에서 Insert > Slice를 선택하십시오 (또는 S를 누르십시오). Tool을 선택하고 캔버스의 아무 곳이나 클릭 앤 드래그하여 영역을 정의하십시오. 슬라이스 도구가 활성화됐을 때 레이어를 클릭하면 자동으로 새로운 슬라이스가 레이어 크기만큼 만들어집니다.
레이어 목록에서 다른 레이어와 함께 슬라이스의 이름을 지정할 수 있습니다. 선택된 레이어로 Command-R을 누르기만 하면 됩니다. 슬라이스를 내보낼 때 만들어지는 파일 이름이 되기 떄문에 적절한 이름을 지정하는 것이 중요합니다.

팁 : 레이어 이름에 슬래시 (/)를 포함하면 해당 슬라이스가 새 폴더에 배치됩니다. 예를 들어, 슬라이스 foo / bar.png의 이름을 지정하면 먼저 foo라는 폴더에 bar.png라는 이미지가 만들어집니다.
여러 크기
슬라이스을 선택하면 Inspector 하단에서 내보내기 설정을 변경할 수 있습니다. 하나의 슬라이스를 크기나 포맷이 다른 여러 이미지로 내보낼 수도 있습니다. 새 축척을 포함하려면 추가 버튼을 클릭하기 만하면됩니다. Exportable 액션 만들기와 마찬가지로 기본 프리셋에 정의된 모든 비율과 설정이 적용된 슬라이스가 나타납니다. 이 기능에 대해 자세히 알아 보려면 위의 Exportable하게 만들기 섹션을 읽으십시오.
투명한 픽셀 다듬기
Inspector의 이 옵션은 이미지에서 불필요한 '텅 빈' 픽셀을 자동으로 제거하는데 사용됩니다. Transparent Pixels (투명 픽셀 다듬기) 체크박스을 선택하면 아래의 내보낼 이미지를 미리 볼 수 있습니다.

투명도는 흰색과 회색 바둑판으로 표시되며, 이미지보다 슬라이스가 클 경우 이 패턴이 이미지 주변에 표시됩니다. Trim를 선택하면 내보내기의 높이와 너비가 이미지의 크기에 맞게 변경됩니다. 내보내기 미리보기는 내보낸 이미지의 모양을 정확하게 보여줍니다. 스케치 창 바깥으로 이미지를 클릭 앤 드래그하여 내보낼 수도 있습니다.
그룹 내용만 내보내기
슬라이스가 그룹에 있으면 Export Group Contents Only 체크박스가 활성화됩니다. 선택하면 주석에 표시된대로 작동합니다. 슬라이스는 해당 그룹 내의 컨텐츠만 내보내므로 캔버스에서 위/아래의 모든 컨텐츠가 제외됩니다. 이 기능은 배경 레이어에서 아이콘을 분리하지 않고도 디자인에서 아이콘을 내보내는 데 적합합니다.
배경색
Background Color 체크박스를 선택하면 슬라이스에서 투명도가 제거되고 선택한 색상으로 바뀝니다. 투명도를 지원하지 않는 JPG와 같은 파일 포맷으로 내보낼 때 이상적입니다.
파일 포맷
이미지를 스케치에서 비트 맵 및 벡터 파일 포맷을 비롯한 다양한 파일 포맷으로 내보낼 수 있습니다. 각각어디에 쓰이냐에 따라 장점이 있습니다.
스케치 문서는 자체 스케치 파일 포맷(.sketch)로 저장되며 스케치 앱에서만 열 수 있습니다. 여러 프로그램에서 볼 수 있는 있도록 특정 영역을 공유하려는 경우, 아래 파일 유형 중 하나로 내보내야합니다.
비트 맵
비트 맵 (또는 래스터) 이미지는 그리드에 기반한 픽셀로 구성되며 광범위하게 지원해 가장 일반적인 이미지 유형입니다. 이것들은 스케치와 같은 앱에서 열 때 'Flattend'를 의미하며 하나의 레이어로 표시됩니다.

PNG : 고품질 이미지를 출력하고 투명도를 지원합니다.
JPG : 큰 이미지 및 사진과 관련된 일반적인 파일 포맷입니다. 저장 시 품질을 조정할 수 있지만 투명한 픽셀은 지원하지 않습니다.
TIFF : 여러 가지면에서 PNG와 비슷하지만 일부 운영 체제에서 사용됩니다. 이러한 이미지는 압축되지 않으므로 파일 크기가 커지는 경향이 있습니다.
WebP : 웹상의 이미지를 작고 풍부하게 만들어주는 새로운 이미지 포맷. 투명도가 지원되며 파일 크기는 PNG보다 3 배 작을 수 있습니다.
벡터
벡터 이미지는 오랫동안 존재했지만 화면 해상도의 수에 따라 웹, 모바일 디자인에서 보다 일반적으로 사용되고 있습니다. 레이어와 패스는 깨지지 않고 품질을 유지하면서 자유롭게 편집하고 크기를 조절 할 수 있습니다.

SVG : 코드로 이미지를 생성할 때 웹에서 가장 일반적으로 사용되는 포맷인 "Scalable Vector Graphic"의 약자입니다. 스케치에서 직접 복사 할 수 있는 코드.
PDF : 일반적으로 문서와 관련된 가장 일반적인 유형의 벡터 포맷입니다. 애플리케이션에서는 이제 에셋과 간단한 아이콘을 PDF 이미지를 사용하는 경향이 있습니다.
EPS : 일반적으로 일부 추가 기능을 포함하는 인쇄 디자인에 사용되지만 주로 PDF로 대체 된 파일 포맷입니다.

참고 : 스케치에서 적용할 수 있는 일부 효과는 모든 벡터 파일 포맷에서 지원하지는 않습니다. PDF의 투명 그라디언트나 SVG의 내부 및 외부 테두리 등
아트보드 내보내기
스케치의 아트보드는 직접 내보낼 수도 있어 따로 슬라이스 레이어를 문서에 추가해야 할 필요가 없습니다. "Exportable"을 클릭하고 내보내기 설정만 하면됩니다. 이를 적용하면 내보내기 창이 뜹니다.
아트보드 내보내기는 디자인를 공유 할 때 특히 유용합니다. 개발자와 동료는 최종 제품이 구현을 찾는 방법을 알 수 있습니다. 아트보드를 다른 레이어와 함께 개별적으로 내보내는 것 외에도 모든 아트보드를 하나의 PDF 파일로 내보낼 수 있어 디자인을 프리젠테이션 할 때 이상적입니다. 이렇게하려면 메뉴에서 Share > Export Artboards to PDF...를 선택하십시오. 아트 보드 내보내기 기본 설정을 변경하여 아트보드가 표시되는 순서를 정할 수 있습니다.
코드 내보내기
일부 디자인을 코드로 내보낼 수 있습니다. CSS, SVG. 웹 디자이너에게 이상적입니다.
CSS 속성
캔바스에서 원하는 수만큼 요소를 선택하면 Control 키를 누른 채로 바로 가기 메뉴에서 Copy CSS Attributes를 선택하십시오. 채우기 (그라디언트 포함), 테두리, 그림자, 모서리 반경 및 전체 텍스트 스타일과 같이 레이어에 있을 수 있는 스타일 정보가 포함된 코드를 클립 보드로 직접 복사합니다. 다음은 간단한 모양의 출력입니다.

/* Rectangle: */ background: #6DD400; border-radius: 5px;
SVG 코드
마찬가지로 SVG 코드 내보내기도 같은 방식으로 작동합니다. 레이어를 Control- 클릭(우클릭)하고 바로 가기 메뉴에서 SVG 코드 복사를 선택하십시오. 이렇게 하면 전체 코드가 복사되며 텍스트 편집기로 열 필요 없이 시간을 절약 할 수 있습니다.
인쇄
스케치의 아트보드와 슬라이스을 인쇄 할 수도 있습니다. 메뉴에서 File > Print를 선택하면 아트보드 목록이 표시됩니다. 슬라이스를 인쇄 할 수 있습니다.
표시된 것 중 하나를 클릭하면 인쇄 설정을 구성 할 수있는 시스템의 창이 뜹니다.

참고 : A4, A5 및 A6 용지 크기의 아트 보드 사전 설정이 있습니다.
