레이어 그룹화
도형, 이미지 및 텍스트와 같은 일반적인 레이어 뿐만 아니라 레이어 그룹화를 지원합니다.이 레이어 그룹화는 문서를 체계적으로 정리할 수 있도록 다른 레이어를 포함할 수 있습니다.
이 섹션에서는 레이어 그룹화 및 캔버스에 영향을 주는 아트 보드와 문서에 여러 캔버스를 만들어주는 페이지를 살펴볼 것입니다.
그룹
문서의 레이어를 그룹화하여 하나의 레이어로 표시 할 수 있습니다. 일반 레이어와 마찬가지로 그룹을 이동하고 크기를 조정할 수 있으며 내부의 컨텐츠에 접근할 수 있습니다.
레이어를 그룹화하면 문서를 체계적으로 정리할 수 있습니다. 함께 표시할 레이어를 그룹화하여 그룹 가시성을 전환하거나 불투명도 및 혼합 모드같은 속성을 조정할 수 있습니다. 그룹을 중첩할 수 있으므로 다른 그룹을 포함할 수 있어 레이어 목록을 편하게 정리할 수 있으며 더 빠르게 탐색 할 수 있습니다. 그룹을 만들려면 하나 이상의 레이어를 선택한 다음 도구 모음에서 그룹 아이콘을 클릭하면 스케치가 해당 항목으로 새 그룹을 만듭니다. 또는 Ctrl 키를 누른 상태에서 바로 가기 메뉴를 통해 레이어를 그룹화하거나 Command-G 키를 눌러 레이어를 그룹화 할 수 있습니다. 레이어 목록을 사용하여 레이어를 그룹으로 드래그하거나 그룹 밖으로 드래그할 수도 있습니다.
그룹 편집
그룹을 만들었으면 캔버스에서 그룹을 두 번 클릭하거나 Enter 키를 눌러 그룹의 내용을 편집할 수 있습니다. 그룹 외부의 개체를 클릭하면 그룹에서 벗어나 문서의 다른 레이어를 다시 선택할 수 있습니다.
클릭 스루
기본적으로 그룹을 처음 두 번 클릭하지 않으면 그룹 안의 레이어를 직접 선택할 수 없습니다. 그러나 Command 키를 누르고 있으면 스케치를 통해 그룹에 포함된 레이어를 선택할 수 있습니다. 이 동작을 피하고 클릭시 그룹의 내용을 자동으로 선택하려면 스케치 환경 설정에서 캔버스 탭 아래의 "Enable click-through for new groups"를 선택할 수 있습니다. 특정 그룹에만 클릭 스루가 적용되도록하려면 그룹을 선택하고 속성에서 "Click-through when selecting"를 선택하십시오.
아트보드
아트보드에서는 스케치의 무한한 캔버스에서 고정된 프레임을 만들 수 있으며 자유롭게 구성할 수 있습니다.
반응형 웹이나 모바일 앱의 여러 뷰를 디자인할 때 크기에 맞게 다양한 화면을 아트보드로 추가할 수 있습니다. 아트보드를 편리한 박스로 사용할 수 있습니다.

아트보드는 특별한 그룹처럼 작동합니다. 주된 차이점은 캔버스에 표현된다는 것입니다. 내용을 보려면 두 번 클릭 할 필요가 없으며 내부 컨텐츠에 맞게 크기가 조정되지 않습니다.
아트보드는 캔버스에서 이름을 클릭하거나 아트보드를 클릭 앤 드래그해 선택할 수 있습니다. 한 번에 여러 장의 아트보드를 선택할 수 있는 편리한 방법입니다.
아트보드 추가하기
아트보드를 만들려면 툴바나 메뉴에서 Insert> Artboard를 선택하십시오 (또는 A를 누르십시오). Inspector는 목록 상단의 드롭 다운 버튼을 클릭하여 카테고리를 변경할 수 있는 프리셋 목록과 가로 및 세로 방향 토글을 표시합니다. 여기에 사용자 정의 크기를 추가 할 수도 있습니다.
목록에서 사전 설정을 클릭해 캔버스에 삽입하면됩니다. 레이어를 선택하면 사전 설정 목록의 맨 위에 선택 영역에 맞게 아트보드를 만들 수 있습니다. 하나의 아트보드를 삽입하고 같 종류의 아트보드를 더 추가하고 싶다면 Command-D (복제)를 누르면 됩니다. 사용자 정의 사전 설정을 추가하려면 속성의 오른쪽 하단에 있는 추가 버튼을 클릭하고 이름과 수치를 입력하십시오.
아트보드 이동
아트 보드를 클릭하고 드래그하여 이동할 수 있습니다 (내용이 포함되어 있지 않은 경우). 그 이유는 실수로 아트보드를 이동하지 않고도 콘텐츠를 선택하고 이동할 수 있기 때문입니다. 아트보드와 모든 레이어를 이동하려면 캔버스에서 아트보드를 클릭하고 드래그합니다. 또는 레이어 목록에서 아트 보드의 이름을 클릭한 뒤 내부로 드래그하여 위치를 변경할 수 있습니다. 아트보드를 선택한 상태에서 Inspector의 위치 값을 사용하여 이동하거나 화살표 키를 사용하여 특정 방향으로 조금씩 이동할 수 있습니다.
아트보드 크기 조정
아트보드의 크기 조정은 다른 레이어와 같은 방식입니다. 선택 핸들을 클릭하고 드래그하거나 Inspector에서 높이와 너비 값을 편집합니다. 하지만 아트보드는 특수한 유형의 레이어이기 때문에 내용을 수정할 수 있는 몇 가지 고유 한 옵션이 제공됩니다. Inspector의 "Resize on contentize"옵션은 레이어에 크기 조절 제약 조건을 적용한 후에 완벽하게 작동합니다. 이 옵션을 선택하면 아트보드 내의 레이어가 아트보드와 함께 크기가 조정되므로 크기가 다른 여러 디스플레이에서 인터페이스를 디자인할 때 이상적입니다.하지만 예측하지 않은 상황을 피하기 위해 레이어 제약 조건이 적용되었는지 확인하십시오!
아트보드의 크기를 임의의 크기로 조정할 수 있을뿐만 아니라 사전 설정 중 하나를 사용하여 빠르고 쉽게 크기를 조정할 수 있습니다. 사전 설정 레이블을 클릭하면 저장된 모든 아트보드 보드 사전 설정을 포함하는 팝업 표시가 나타나고 새 크기를 선택할 수 있습니다. 아트보드에서 가로까지 (또는 그 반대로) 아트보드 보드의 방향을 완전히 조정할 수 있으며 크기 전환기 팝업 메뉴의 사전 설정 값도 변경됩니다.
크기에 맞게 조정
크기에 맞추기 옵션은 아트 보드의 내부에 있는 레이어의 높이와 너비에 맞게 아트보드의 크기를 자동으로 조정합니다. 스크롤 할 때 높이가 필요한 모바일 뷰를 디자인 할 때 유용합니다.
아트보드 전환
Function-Left Arrow를 눌러 왼쪽으로 이동하거나 Function-Right Arrow를 눌러 오른쪽으로 이동하여 문서의 아트보드간에 빠르게 이동할 수 있습니다.
아트보드 삭제
아트보드를 제거하려면 Backspace 키를 누르면 해당 내용이 삭제됩니다. 아트보드만 삭제하려는 경우 툴바에서 그룹 해제를 선택하거나 Shift-Command-G를 누릅니다.
그리드와 눈금자
각 아트보드는 독립적인 캔버스로 처리되어 툴바의 View 팝업 메뉴를 통해 적용 할 수있는 자체 격자와 눈금자가 포함됩니다. 이것들은 Canvas의 눈금자와 눈금에 독립적입니다.
템플릿
실제로 사용되고 있는 여러 아트보드의 좋은 예를 보려면 스케치에서 제공하는 기본 템플릿을 확인하십시오. 메뉴에서 File> New From Template를 선택하십시오. iOS 앱 아이콘 템플릿을 선택하면 일반적인 크기의 아트보드가 표시됩니다.
페이지
문서에는 여러 페이지가 포함될 수 있습니다. 새 페이지를 추가하려면 레이어 목록 맨 위의 페이지 목록 표시 버튼을 클릭하고 추가 버튼을 클릭합니다. 페이지를 사용하면 여러 캔버스를 문서에 저장할 수 있으며 여러 이점이 있습니다.
예를 들어 문서에 새 심볼을 추가하면 심볼 페이지로 자동으로 정리할 수 있습니다. 이 페이지는 일반 페이지와 동일하지만 재사용하는 모든 요소를 한 곳에 저장합니다.

여러 페이지를 사용하는 또 다른 이유는 성능과 관련이 있습니다. Sketch는 페이지에서 12 개의 아트보드를 쉽게 처리 할 수 있지만 내용, 크기 및 개수에 따라 여러 페이지로 작업을 분할하는 것이 좋습니다.

여러 페이지를 사용하는 다른 권장사항은 스타일 가이드가 포함된 웹 디자인의 서로 다른 페이지를 복제하는 것입니다. (페이지를 Command- 클릭하고 팝업 메뉴에서 복제를 선택하여 페이지를 복제 할 수 있음). 남는 캔버스 공간을 아이디어를 실험하기 위한 영역으로 사용할 수 있습니다.
페이지 전환
Function-Up Arrow 를 눌러 위로 이동하거나 Function-Down Arrow 를 눌러 페이지 목록 아래로 이동하여 문서의 페이지간에 빠르게 전환 할 수 있습니다.

레이어 목록 맨 위에있는 페이지 이름을 클릭하면 대상을 선택할 수있는 팝업 메뉴가 나타납니다.
다중 선택
또한 Command-clicking하여 페이지 목록에서 여러 페이지를 선택할 수도 있습니다. 여러 페이지를 선택하면 효율적으로 삭제 및 재정렬할 수 있습니다.
