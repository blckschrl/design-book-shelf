텍스트
스케치는 macOS의 기본 글꼴 렌더링을 사용해 텍스트가 아름답게 보입니다. 네이티브 글꼴 렌더링의 이점은 웹 사이트 디자인을 만들 때 텍스트가 정확하게 보이는 것입니다. 또한 텍스트 스타일을 지원해 여러 텍스트 레이어가 동일한 서체, 크기, 색상 등을 공유할 수 있습니다.
텍스트 추가하기
툴바의 Insert 팝업 메뉴에서 텍스트 도구를 선택하여 텍스트를 추가 할 수 있습니다 (또는 T 키를 누름). 포인터가 텍스트 입력 툴 아이콘으로 바뀌고 캔버스의 아무 곳이나 클릭하여 그 지점에 텍스트 레이어를 삽입 할 수 있습니다. 새 텍스트 레이어에 "Type something"가 표시되며 새 텍스트를 넣을 준비가 되었습니다.

클릭 앤 드래그해 고정된 크기의 텍스트 박스를 만들 수도 있습니다. 텍스트가 박스 안에 들어가기에는 너무 커지면 내용에 맞게 너비가 증가하는 일반 텍스트 박스와는 달리 새 줄이 만들어집니다.
누락된 글꼴
인터넷에서 다운로드되었거나 동료로부터 받은 Sketch 파일로 작업 할 때 시스템에 없는 글꼴이 포함되어 있을 수 있습니다. 스케치는 이 경우를 처리하도록 설계되었으며, 창의 오른쪽 상단 모서리에 디자인에 사용자가 보유하지 않은 글꼴을 사용한다는 경고를 표시합니다.
버튼을 클릭하면 문서에 사용된 누락된 글꼴 목록이 포함 된 창이 표시되고 대체 글꼴 및 글꼴 두께를 선택하여 대체 할 수 있습니다. 또는 이 목록에서 누락된 글꼴을 설치할 수 있습니다.
서식이 있는 텍스트
스케치는 다른 응용 프로그램에서 붙여 넣은 서식있는 텍스트를 표시하는 기능을 지원합니다. 예를 들어 Safari에서 웹 사이트의 일부 텍스트를 선택하여 복사 한 경우 스케치에서 Edit › Paste › Paste as Rich Text (또는 Option-Shift-Command-V 누르기)를 선택하십시오. 새로 붙여 넣은 텍스트 레이어에는 설정된 글꼴, 크기 및 색상의 단어가 포함됩니다.
텍스트 크기 조절하기
캔버스에서 텍스트 박스의 크기를 조정하면 텍스트 크기가 변경되지 않습니다. 텍스트 크기를 변경하려면 Inspector에서 텍스트 크기값을 편집해야합니다.
텍스트 Inspector
표준 레이어 속성 아래에는 공유 텍스트 스타일 영역이 있습니다.

그 아래에서 원하는 다양한 텍스트 속성을 선택하고 적용할 수 있습니다. 글자체 (글꼴 군), 무게 (예 : 밝음, 굵게, 기울임 꼴) 및 글꼴 크기와 같은 또한 텍스트 정렬을 조정하고 레이어가 고정 폭인지 자동 폭인지 결정하고 문자, 줄 및 단락 간격을 조정할 수 있습니다.
텍스트 색상
텍스트를 편집할 때 텍스트의 하위 범위에 특정 색상을 적용 할 수 있습니다. 글꼴 크기와 옵션 버튼 사이의 색상 버튼를 사용하면됩니다.

그래디언트와 같은 텍스트 레이어에 일반 채우기 (Full Fill) 스타일을 적용할 수도 있습니다. 그러나 여기에 적용된 채우기는 전체 텍스트 레이어에 적용되며 위에 있는 색상 박스에서 설정한 항목보다 우선 적용됩니다.
텍스트 옵션
옵션 버튼을 클릭하면 밑줄 및 취소 선과 같은 텍스트 장식을 적용할 수 있는 팝업과 여러 줄이 있는 텍스트 레이어를 목록으로 변환 할 수 있는 옵션 및 텍스트 선택을 변형하는 옵션이 표시됩니다.

텍스트 변형 팝업 메뉴에서 완전히 비파괴적인 방식으로 텍스트가 어떻게 보이는지 조정할 수 있습니다. 즉, 텍스트를 다시 입력 할 필요 없이 대문자 또는 소문자로 스타일을 지정할 수 있습니다.
정렬
스케치에서 텍스트를 세로로 정렬 할 수도 있고 가로로 정렬 할 수도 있습니다. 텍스트 레이어의 정렬에 대한 변경 사항을 보려면 텍스트 박스를 자동 또는 고정으로 설정할 수 있습니다.

맞춤을 자동으로 설정하면 텍스트 박스의 너비가 입력 한 텍스트에 맞게 가능한 넓게 펼쳐지며 입력할 때 가로로 맞춰집니다.

맞춤이 고정으로 설정된 경우 캔버스에 표시되는 핸들을 클릭하고 드래그하여 텍스트 레이어의 높이와 너비를 조정할 수 있습니다. 텍스트를 입력 할 때 단어가 줄바꿈되고 단어가 나머지 너비에 맞지 않으면 새 줄이 만들어집니다.

수평 정렬을 조정할 수 있을뿐만 아니라 텍스트 레이어에 고정 높이를 지정하면 텍스트를 위쪽, 중간 또는 아래쪽으로 세로로 정렬할 수 있습니다.
라인 높이
스케치는 선 높이를 조정할 때마다 단락에 대한 일관된 기준선 간격을 유지합니다. 텍스트 레이어의 서체 또는 글꼴 크기를 변경하면 레이어가 다시 배치되어 첫 번째 기준선이 항상 그대로 유지됩니다. 일관된 기준선 오프셋은 고정된 선 높이를 갖는 한 다양한 글꼴을 사용할 때도 단락간에 유지됩니다. 예측 가능한 수직적인 리듬이 아름답게 구현됩니다. 고정된 선 높이가 설정되지 않은 경우 글꼴 자체에 지정된 높이를 사용합니다.
새 텍스트 레이어를 만들 때 이전에 설정된 내용을 상속하지 않고 "자동" 줄 높이를 사용합니다. 텍스트 레이어의 자동 줄 높이는 동일한 값의 고정 된 줄 높이와 구별되며 Inspector는 자동 값을 자리 표시 자로 표시합니다.
텍스트 스타일
많은 텍스트 레이어가 포함된 인터페이스 또는 웹사이트를 디자인 할 때 많은 레이어에 동일한 텍스트 속성이 포함됩니다. 이러한 레이어에 적용할 텍스트 스타일을 정의할 수 있어 스타일을 쉽게 업데이트 할 수 있습니다.
참고 : 텍스트 스타일은 문서 단위로 존재합니다. 문서 간에는 공유할 수 없지만 문서의 모든 페이지와 아트보드간에 사용할 수 있습니다.
스타일 만들기
텍스트 스타일을 만들려면 텍스트 레이어를 선택하고 메뉴에서 Layer> Create Shared Style을 선택하십시오. Inspector가 업데이트되어 선택한 레이어의 속성을 스타일이 적용되어 만들어진 새 텍스트 스타일을 보여줍니다.
글꼴, 정렬 등과 같이 사용자가 수정하는 텍스트 스타일의 모든 부분은 선택한 레이어에 자동으로 업데이트됩니다. 이러한 변경 사항이 적용된 다른 텍스트 레이어를 동일한 텍스트 스타일로 업데이트하려면 스타일 미리보기 창에서 업데이트 버튼을 클릭하십시오.

변경 후에 특정 레이어의 스타일을 업데이트하지 않으려면 Option 키를 누른 상태에서 업데이트 위치에 나타나는 Revert 버튼을 클릭합니다.
새 레이어로 삽입
기존 텍스트 스타일을 텍스트 레이어에 적용하려면 해당 텍스트 레이어를 선택하고 속성에서 "No Text Style"을 클릭하여 문서에 정의된 텍스트 스타일을 보여주는 팝업 메뉴를 표시하십시오.

이미 스타일이 적용된 텍스트 레이어를 삽입하려면, 툴바에서 팝업 메뉴의 Insert> Styled Text를 선택하고 사용할 텍스트 스타일을 선택하십시오. 그런 다음 다른 텍스트 레이어를 삽입하는 것처럼 캔버스를 클릭하십시오.
패스 위의 텍스트
텍스트 레이어를 벡터 경로에 적용할 수 있습니다.
이 효과를 만들려면 텍스트 레이어와 텍스트를 배치 할 패스가 있는 도형 레이어가 필요합니다. 이 레이어는 레이어 목록의 텍스트 레이어 아래에 나타나야합니다. 선택한 텍스트 레이어를 사용하여 메뉴에서 Text> Text on Path를 선택한 다음 레이어를 벡터 모양으로 이동합니다. 해당 위치로 드래그해 '스냅'해야 합니다. 텍스트 레이어를 패스에 물리기를 멈추려면 레이어 목록에서 해당 위치의 순서를 바꾸거나 Text> Text on Path를 다시 눌러 취소하십시오.
윤곽선으로 변환
텍스트 레이어를 벡터 모양으로 변환하려면 메뉴에서 Layer › Convert to Outlines을 선택하거나 Shift-Command-O를 누릅니다. 그러면 다른 모양처럼 개별 패스와 점을 편집 할 수 있습니다.

경고 : 사용 시 주의하시기 바랍니다. 텍스트를 외곽선으로 변환하면 텍스트를 편집 할 수 없습니다. 많은 양의 텍스트를 변환하면 패스 숫자와 Boolean Operation 때문에 성능이 크게 떨어집니다.
이미지
이미지 또는 비트맵은 Sketch에서 지원되는 레이어 유형 중 하나입니다. 스크린샷이나 사진과 같은 다양한 형태를 취할 수 있으며 벡터 모양처럼 자유롭게 편집하거나 조작 할 수 없습니다.
일반적으로 필요하지는 않지만 스케치의 모든 레이어를 단일 비트 맵으로 변환 할 수 있습니다. 메뉴에서 Layer › Flatten Selection to Bitmap을 선택하십시오.
이미지 크기 줄이기
Sketch에서 작업 할 때 사진이나 스크린 샷과 같이 디자인에 이미지를 추가하는 것이 너무나 일반적입니다. 더 많이 추가할수록 Sketch 문서는 더 커질 것입니다.

전체 파일 크기가 증가함에 따라 스케치가 문서의 특정 부분을 미리 보는 데 시간이 오래 걸릴 수 있습니다. 따라서 파일에서 사용 중인 이미지의 크기를 줄이는 것이 좋습니다. 메뉴에서 Layer › Image › Minimize File Size를 선택합니다. 스케치는 파일 크기를 줄이기 위해 크기가 조정되고 크기가 조정된 이미지 (채우기의 일부 또는 이미지 레이어에 관계없이)를 최적화하여 공유하기 쉽게 만들고 성능을 향상시킵니다.
이미지 크기 줄이기는 파일 크기를 메가 바이트 단위로 줄일 수 있는 매우 편리한 방법입니다.
이미지 대체하기
스케치에서 이미지를 쉽게 바꾸려면 단순히 이미지를 선택하고 Layer › Image › Replace…를 선택하고 대화 박스에서 대체를 선택하십시오. 이 방법은 스타일 과 크기를 유지합니다. 또는 이미지를 Control- 클릭하고 바로 가기 메뉴에서 Replace Image…를 선택합니다.

이미지를 원래 크기로 되돌리려면 메뉴에서 Layer › Image › Set to Original Dimensions을 선택합니다.
비트맵 편집
스케치에는 일반적인 비트맵 편집 기능이 포함되어 있어 서로 다른 디자인 툴을 사용하지 않도록 도와줍니다.
이미지 편집을 시작하려면 캔버스에서 이미지를 선택한 뒤 더블 클릭합니다. 그려면 Inspector에 몇 가지 특수한 툴이 표시됩니다. 먼저 캔버스에서 편집하려는 하는 영역을 선택한 뒤 원하는 도구를 선택합니다.

  • Selection : 이미지에서 직사각형 영역을 선택합니다.
  • Magic Wand : 이미지를 클릭하고 드래그하여 영역을 선택합니다. 클릭한 첫 지점부터 멀리 드래그할 수록 tolerance도 커집니다.


    • Shift 키를 누르면서 선택하면 영역이 더 추가되며 Option 키를 눌러 영역을 뺄 수 있습니다. 선택이 완료되면 영역을 잘라내거나 복사하여 새로운 비트맵 레이어에 사용하거나 아래 4가지 도구 중 하나를 선택할 수 있습니다.

  • Invert : 현재 선택한 영역의 반대가 선택됩니다.
  • Crop : 선택한 영역만 표시되도록 레이어를 자릅니다.
  • Color : 선택한 영역을 선택한 색상으로 채웁니다. 표준 색상 선택 도구가 나타나 색상을 선택할 수 있습니다.


    • 편집이 끝나면 이미지 바깥 쪽 아무 곳이나 클릭하거나 엔터 키 또는 Escape 키를 눌러 종료할 수 있습니다.
    색상 조정
    이미지의 색상을 조정하려면 Inspector의 Color Adjust 패널을 사용하면됩니다. 이 패널에서 색조, 채도, 밝기 및 대비를 변경할 수 있습니다.
    참고 : 이것은 비파괴적인 효과이므로 나중에 언제든지 값을 변경할 수 있습니다.
